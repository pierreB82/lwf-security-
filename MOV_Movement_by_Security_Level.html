Movement by Security Level

Assign a security colour code to each area or route and document the movement rules that apply.

Colour Code Reference

● Assign a colour code to each area based on the security situation observed during the last 3 months. Update whenever a significant event occurs.
GREEN - Stable
Stable environment. No significant security incident targeting NGOs in the last 3 months.
YELLOW - Unstable
Unstable environment. Serious incidents targeting NGOs in the last 3 months, or risk of armed incident (lean season, transhumance, inter-community tensions...).
RED - Restricted
Very unstable. Serious and repeated incidents. No guarantee from armed actors. Movements prohibited except by special authorisation from the Country Representative.
